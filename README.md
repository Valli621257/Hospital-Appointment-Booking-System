# Hospital-Appointment-Booking-System

# Project Overview
The Hospital Appointment Booking System is a full-stack web application that allows patients to book doctor appointments online. It helps hospitals manage patients, doctors, and appointments efficiently in a digital way.

# Tech Stack

#Backend:
- Java
- Spring Boot
- Spring Security (JWT)
- REST API
- MySQL

### Frontend:
- React JS
- HTML
- CSS
- JavaScript

---

#Features

-  User Registration & Login (Patient/Admin)
-  JWT Authentication & Security
-  Doctor Management (Add/View/Delete)
-  Patient Management
-  Book Appointment
-  Cancel Appointment
-  View Appointments
-  Admin Dashboard

# Project Structure

- `hospital-backend` → Spring Boot backend (APIs, database logic)
- `hospital-frontend` → React frontend (UI)

# How to Run the Project

### Backend (Spring Boot):
1. Open backend in IDE (IntelliJ / Eclipse)
2. Configure MySQL database in `application.properties`
3. Run the Spring Boot application

### Frontend (React):
```bash
npm install
npm start
